# LLM Governance Engine (2026)

An Enterprise evaluation framework for balancing Frontier intelligence with Local efficiency.

## 🛠️ Tech Stack
- **Frontend:** Angular (RxJS, Signals)
- **Backend:** FastAPI (Python 3.12+)
- **Database:** PostgreSQL (with SQLAlchemy ORM)
- **Orchestration:** Docker Compose & Kubernetes (AKS/EKS)
- **AI Models:** OpenAI/Claude (Cloud) & Ollama (Local)
